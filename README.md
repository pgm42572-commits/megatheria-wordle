# 🦁 MEGATHERIA WORDLE

**Adivina la fauna de Luján** - Juego educativo tipo Wordle/Metazooa

---

## 🎮 Sobre el Juego

MEGATHERIA WORDLE es un juego educativo interactivo que combina la mecánica de Wordle con fauna reali de Luján, Buenos Aires. Debes adivinar un animal secreto comparando atributos científicos.

### Características

✅ **34 especies** verificadas científicamente  
✅ **Sistema de 10 intentos** con comparación visual  
✅ **Autocompletado dinámico** para búsqueda  
✅ **Ficha técnica educativa** estilo museo paleontológico  
✅ **Estadísticas personales** de partidas  
✅ **Compartir puntajes** por redes sociales  
✅ **Fauna Lujanense** panel lateraldesde
✅ **Imágenes de Wikimedia Commons** - URLs verificadas

---

## 🎯 Cómo Jugar

1. **Adivina el animal secreto** escribiendo nombres en la barra de búsqueda
2. **Analiza las pistas**: Cada atributo se compara con el animal secreto
   - 🟢 **Verde** = Coincide exactamente
   - 🟡 **Amarillo** = Parcialmente relacionado (puede usarse en futuras mejoras)
   - ⚫ **Gris** = Diferente
3. **Tienes 10 intentos** para acertar
4. **Desbloquea la ficha técnica** al ganar o perder

---

## 📊 Sistema de Atributos

Cada animal tiene 4 atributos científicos:

- **Época**: Prehistórico o Actual
- **Grupo**: Clasificación evolutiva (Xenartro, Carnívoro, Litopterno, etc.)
- **Dieta**: Herbívoro, Carnívoro u Omnívoro
- **Ambiente**: Pastizal, Bosque o combinaciones
